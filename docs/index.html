Abstract

Recent studies have proposed to estimate audio synthesizer parameters using deep learning models. In some of these, the models are trained to match the synthesizer's output to the target signal. This requires the synthesizer to be implemented in a differentiable manner, hence those approaches are named ``differentiable digital signal processing'' (DDSP). However, these algorithms are usually not injective functions w.r.t. their parameters: multiple different parameter combinations can generate the same output. This is in contradiction with most approaches which output a single parameter combination to synthesize a given sound. In this work, we propose a probabilistic model that learns all valid parameter combinations. For this, we use the normalizing flows (NF) framework along with mixtures of Gaussians (MoG) to model the target parameter distribution. We optimize our model using the sum of two loss terms: an audio reconstruction loss computed using differentiable signal processing (so that the parameters accurately reproduce the target sound) and an entropy regularization, which encourages the model to output diverse parameter combinations. We evaluate our approach on frequency modulation synthesis parameter estimation.
