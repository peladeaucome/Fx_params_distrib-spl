♯: LTCI, Télécom Paris, France.

♭: IBISC, Univ. Evry Paris-Saclay, France.

Abstract

Audio synthesizer parameter estimation is intrinsically multimodal: different parameter configurations can produce the same target sound. However, most estimators return a single parameter set, which hides this ambiguity. We propose a probabilistic approach that estimates the conditional distribution of synthesizer parameters. Our model uses a mixture of Gaussians (MoG) base distribution transformed by normalizing flows (NF). It is trained with an audio reconstruction loss computed through differentiable signal processing together with an entropy regularization term that encourages diverse valid solutions. We use simulated annealing to control the trade-off between reconstruction accuracy and parameter diversity during training. We evaluate our approach on a three-branch frequency modulation (FM) synthesizer using realistic DX7 presets, and compare it with deterministic and flow-matching baselines. The results show that MoG-based NF improve the trade-off between audio reconstruction accuracy and parameter diversity, and can recover meaningful symmetries in the FM parameter space.
